conn¶

Lazily load and return the raw connection from threadlocals.

Connect to the LDAP server specified in settings and bind using the GSSAPI protocol. The requisite KRB5CCNAME environmental variable should have already been set by the SetKerberosCache middleware.

intranet.db.ldap_db.close_ldap_connection(sender, **kwargs)[source]¶

Closes the request’s LDAP connection and clears up thread locals.

Listens for the request_finished signal from Django and upon receipt, unbinds from the directory, terminates the current association, and frees resources.

It would be nice if we could leave the connections open in a pool, but it looks like rebinding on an open connection isn’t possible with GSSAPI binds.
