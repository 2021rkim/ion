Welcome to the TJ Intranet

The TJ Intranet allows students, teachers, and staff at The Thomas Jefferson High School for Science and Technology (TJHSST) to access student information, manage activity signups, and view information on news and events.



Intranet is a student-built project that was originally built to support the Eighth Period program at TJHSST, but now does so much more. For the 1999-2000 school year, several students created a web-based solution for activity signups, which was previously a mechanical process. In 2006, Intranet2, codenamed Iodine, was written as a replacement for the previous system that was more efficient and maintainable. This site evolved into a critical part of the school itself, giving access to news, polls, a student directory, and files, among many other functions. After nearly ten years of use, it was deemed necessary to create a new application, using modern web technologies, that could be used on across all forms of devices and platforms. Ion is that project.

Development on Intranet version 3 began in early 2013 and, more than two years later, is now ready for use during the 2015-2016 school year. The application is written using Python and the Django web framework, and on the client side, makes excessive use of responsive elements to allow for the best possible user experience across devices.
